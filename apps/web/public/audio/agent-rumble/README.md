# Agent Rumble audio

`crownfall-battle.ogg` is **Fighting Is Not an Option** by Cleyton Kauffman.

- Source: https://opengameart.org/content/rpg-battle-theme-ii
- License: CC0 1.0 / public domain dedication
- Original filename: `Fighting is not an option.ogg`

ArenaOS renames the file for stable application routing. Combat sound effects are synthesized at runtime with the browser Web Audio API and do not use external samples.
